Welcome to the CAD FEA API tool!

This tool allows you to perform structural analysis on a range of pre-made parametric CAD parts.

Workflow Process

The tool follows a simple process:

  1. The type of part is selected and key driving dimensions are inputted
  2. The part is converted from a CAD file to a mesh file
  3. The structural constraints and loads are specified
  4. The simulation is run, and key outputs (such as maximum stresses and displacements) are displayed
